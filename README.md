# Controls
| Key | Action |
| --- | ------ |
| Arrow Keys | Rotate the camera |
| `r` | Reset the camera rotation |
| `Enter` | Hold to speed up the growth |
| `Space` | Reset the creature |
| `p` | Pause/play the growth |
| `-` | Zoom out |
| `=` | Zoom in |

Also try moving the mouse around and see what happens!
